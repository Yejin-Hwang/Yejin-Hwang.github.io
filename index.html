📌 Thesis · KDD 2026 Under Review
Can Reddit Predict the Stock Market?
💡 The Origin Story

It was early 2025. I was browsing a Korean stock community late at night when I noticed something: a user with a special "stock guru" badge posted about quantum computing stocks going up. People followed. The stock jumped over 200%. Real money moved because of an internet post.

Then I thought about GameStop. The Dumb Money movie. Reddit's r/WallStreetBets — still active, still moving markets. The question formed itself: what if you could quantify that signal?

GameStop's 1700% surge (Jan–Mar 2021) · Beyond Meat 1300% spike — both driven by social media momentum
🔬 The Research Question

Can a pre-trained Transformer model (Temporal Fusion Transformer) trained on historical stock prices perform better when you add social media sentiment scraped from Reddit — analyzed through FinBERT, a finance-tuned BERT model?

Hypothesis Social media sentiment from WallStreetBets contains predictive signal for short-term stock price movement that deep learning models can learn to use.
⚙️ Research Pipeline
Full research pipeline: Yahoo Finance API + Reddit API → Feature Engineering → Modeling (TFT) → Evaluation

Reddit posts from r/WallStreetBets were scraped via the Reddit API and analyzed using FinBERT (finance-domain BERT) to extract daily sentiment scores. These were engineered into lag features, rolling statistics, and spike indicators — then combined with OHLCV price data as multimodal input to the Temporal Fusion Transformer (TFT). Three model tiers were compared: univariate baselines (LSTM, PatchTST, TimesFM), TFT without sentiment, and TFT with sentiment.

TFT + Sentiment (green) pulls clearly ahead from Phase 2 onward, reaching ~20% cumulative return by Dec 2025. All other models hover near zero or go negative.

📉 Sharpe Ratio Comparison

TFT + Sentiment achieves a Sharpe ratio of 2.74 — far outperforming TFT Baseline (1.67) and all other models. TimesFM (Zero-shot) goes deeply negative, showing that off-the-shelf foundation models alone aren't enough.

🔮 Predicted vs. Actual Returns — All 10 Equities × All Models

Each cell shows predicted (red) vs. actual (blue) return curves. The bottom two rows — TFT Base and TFT + Sentiment — show the tightest tracking. Green IC scores in TFT + Sentiment confirm sentiment features are genuinely contributing signal, not noise.

🧠 Variable Importance — What the Model Actually Learned

This was the key finding. The TFT's built-in interpretation plot shows that sentiment features dominate: sentiment_lag3, spike_intensity_max_14d, sentiment_mean_14d, and spike_presence_sum_14d all rank in the top 8. The model didn't just perform better — it learned why.

💭 What I Learned

Beyond the technical results, this project taught me that the most interesting data problems live at the intersection of human behavior and quantitative systems. People talking on the internet is data. Communities form opinions that move markets. The challenge — and the opportunity — is figuring out how to listen to that signal responsibly.
